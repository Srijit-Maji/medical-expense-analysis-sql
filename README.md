# 🏥 Medical Expense Analysis using SQL

## 📌 Project Overview

This project analyzes medical expense data using SQL to uncover valuable business insights. The analysis focuses on understanding how factors such as age, gender, BMI, smoking habits, and region influence medical insurance charges.

---

## 🎯 Objectives

- Analyze medical insurance expenses
- Identify factors affecting insurance charges
- Compare expenses across different regions
- Evaluate the impact of smoking on medical costs
- Generate business insights using SQL queries
